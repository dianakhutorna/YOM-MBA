Quick Start
===========

Overview
--------

This quick start follows the documented offline pipeline:

1. Train model
2. Generate batch predictions
3. Serve recommendations via API

Why this order
--------------

Serving depends on artifacts generated by the first two stages.
The system does not perform online ML inference at request time.

Requirements
------------

- Python 3.11+
- Input files prepared:

  - ``training/data/external/commerces.csv``
  - ``training/data/external/products_v2.csv``
  - ``training/data/raw/*.csv``

Setup
-----

.. code-block:: bash

   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

Step 1 — Training
-----------------

.. code-block:: bash

   ./venv/bin/python -m training.src.scripts.run_training_pipeline \
     --config training/configs/training_pipeline.yaml

Main outputs:

- ``training/models/lgbm_ranker.txt``
- ``training/models/lgbm_ranker.features.json``
- ``training/data/interim/orders_sample.parquet``
- ``logs/training_*.log``

Step 2 — Batch scoring
----------------------

.. code-block:: bash

   ./venv/bin/python -m training.src.scripts.generate_predictions \
     --config training/configs/generate_predictions.yaml

Main outputs in ``training/data/interim/``:

- ``predictions.parquet``
- ``popularity_fallback.parquet``
- ``category_fallback.parquet``
- ``global_fallback.parquet``

Step 3 — Local API run (development)
------------------------------------

.. code-block:: bash

   ./venv/bin/python -m training.src.scripts.serve_recommendations_api

Endpoints:

- ``GET /health``
- ``GET /recommendations``
- ``POST /recommendations/multi``
- ``GET /docs``

Recommendation request format:

.. code-block:: text

   GET /recommendations?kioskId=<kiosk_id>&anchorId=<anchor_product_id>&limit=<N>

Verification
------------

.. code-block:: bash

   curl http://localhost:8000/health
   curl "http://localhost:8000/recommendations?kioskId=<kiosk_id>&anchorId=<anchor_id>&limit=10"

Optional quality checks
-----------------------

.. code-block:: bash

   ./venv/bin/python -m pytest training/tests/ -q
   ./venv/bin/python -m training.src.scripts.check_personalization \
     --config training/configs/training_pipeline.yaml --top-k 5 --sample-kiosks 300
   ./venv/bin/python -m training.src.scripts.check_new_vs_repeat --top-k 5 --sample-kiosks 200
